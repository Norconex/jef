This library constitutes the Norconex Java Execution Framework.

Simple steps to get started:

  • Create one or more jobs ({@link com.norconex.jef5.job.IJob}) to accomplish the required tasks.
  • If you have more than one job, assemble/group them together using one or a combination of {@link com.norconex.jef5.job.group.AsyncJobGroup} and {@link com.norconex.jef5.job.group.SyncJobGroup}.
  • Create and configure a job suite ({@link com.norconex.je5f.suite.JobSuite}) with your created jobs.
  • Run your job suite by invoking {@link com.norconex.jef5.suite.JobSuite#execute()}.

The creation and launching of a job suite may look like this: 

    IJob rootJob = new AsyncJobGroup("asyncGroupSample", new IJob[] {
            new MyJobA(),
            new SyncJobGroup("syncGroupSample", new IJob[] {
                    new MyJobB(),
                    new MyJobC()
            }),    
            new MyJobD(),
            new MyJobE()),
    });
    JobSuite suite = new JobSuite(rootJob);
    
    // add any listners/handlers to the suite
    
    suite.execute();
